Porter Example

This is an exmaple app with Porter setup, with its source available at github. Porter is a consolidated solution for browser module authoring which provides:

  • a middleware for browser module transformation,
  • a loader for browser module loading, and
  • a compiler for browser module compilation to make the app production ready.
For brevity, we'll refer to browser modules as components in the rest of this page.

Setup

In app.js, setup the middleware.

In html, use the loader to import the entry component.

In file components/home.js, require all the way.

In file components/stylesheets/app.css, there's @import.
